# Iris-Dataset-Correlation-Linear-Regression-Analysis

This project explores the classic Iris dataset using statistical techniques and linear regression modeling. The focus is on analyzing relationships between flower measurements (sepal length, petal length, petal width) through Pearson correlation and constructing simple linear regression models to predict petal width.
